Semana 5

Criterios para elaborar un mapa de sitio web.

Mapa de sitio

Es una representación visual del sitio web, la cual ayuda a los usuarios a encontrar el contenido de forma ordenada. Enumera las páginas, y otros archivos del sitio web y describe las relaciones que hay entre ellos.

Mapa de sitio visual

Suelen tener la forma de un diagrama jerárquico y ayuda a visualizar el contenido y la jerarquía, al igual que la navegación de un sitio web.

¿Qué incluye un mapa de sitio?

Contiene una página de incio, o una introducción que sirva de resumen para el contenido general de la página. Debe contener características o servicios, precios de los servicios o productos. Un ítem que incluya más información acerca del creador o los creadores del servicio o producto, este ítem puede denominarse "acerca de". También pueden incluirse otrs datos como: Historia de la marca, reseñas, blog, contacto, soporte, pie de página. Esto con el fin de que la página incluya la mayor información posible para que el usuario encuentre todo bien estructurado.

Herramientas online para creación de sitios web

Existen herramientas web que son gratuitas y permiten que se generen páginas sin mucho esfruerzo, ya que ponen a disposición plantillas definidas, para que el usuario solo deba asocial el contenido y los recursos que desee publicar. Esto se puede hacer a traves de servicios web como: google sites, wix, entre otros
