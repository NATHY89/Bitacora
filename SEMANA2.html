Semana 2

¿Qué es un elemento?

Componente básico de un archivo HTML. Se pueden referir como un ladrillo. Usamos elementos para definir la estructura de una página web en un archivo HTML. Elemento= element en inglés

¿Qué son las etiquetas?

Son las formas de escribir los elementos en un archivo, que crean los elementos de la página web. Especifican e tipo de documento que se va a crear. La estructura de la etiqueta es : dentro de dos paréntesis angulares, dentro el tipo de elemento. Son escenciales porque indican donde comienza un elemento y dónde termina con su etiqueta de cierre.

Tipos de etiquetas

  1. Etiqueta de apertura
  2. Etiqueta de cierre

    3. La diferencia es incluir una barra luego del primer paréntesis.

Comenzamos a crear nuestra página

Dentro de esta etiqueta tenemos la etiqueta head y body Dentro de head se puede incluir cualquier archivo externo com css, también se puede personalizar el título de la página. Igualmente si no se incluye, automáticamente se crea. Importante: intentación: es el espacio o la jerarquía de cada elemento. Especificar idioma: para mejorar la accesibilidad de la página. Atributo: especifica algo sobre ese atributo. DAto: Un dato muy chevere, es oprimir alt+Z para adaptar el texto a la pantalla. Si se vuelve a hacer click en este comando puede volver a la función anterior

Buscar en la documentación

Es importante conocer más de la documentación en mozilla firefox, o en gooogle chrome, para conocer más comandos o funciones para crear elementos de la página.

¿Y para crear colores?

Para cambiar el color de los textos usamos la etiqueta style, ya sea junto con la etiqueta h o la etiqueta p, o podríamos usarlo entre paréntesis abriendo y cerrando la etiqueta, para configurarlos todos al mismo tiempo

Para agregar imágenes

Existe la opción de agregar imágenes a nuestra web, ya sea desde una carpeta dentro de la máquina, o directamente usando un enlace de la web. para esto se agrega la etiqueta img combinándola con la etiqueta src

Para agregar comentarios

Existe la opción de crear un comentario para el desarrollador web, pero sin que este comentario aparezca en la página visible al usuario. Esta opción se puede hacer colocando entre paréntesis un signo de exclamación, seguido de dos guiones, el comentario, y lo cerramos con dos guiones más
