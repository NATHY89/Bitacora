Semana 3

Para esta semana ya hemos profundizado más sobre el uso de lenguajes e interfaces de programación para diseño web

Back-end y Front-end

Front-end es conocida como el lado del cliente como el tipo de letra, los colores, efectos del mouse, efectos visuales. Está compuesta por el contenido de la página, y el formato html para la programación, el css para la parte gráfica y js o el servidor web donde va a ser publicada la página. Back-end es el lado del servidor, es una aplicación y base de datos. Acá se manejan lenguajes de programación como frameworks y los tipos de base de datos

Interfaces y lenguaje

Dentro de la estructura de la página web, se utilizan etiquetas incluidas dentro de los símbolos "< >" y están ordenados según una jerarquía..

Contenedores

Son recuadros o espacios rectangulares que pueden colocarse en cualquier parte de la página, y su principal utilidad es visual, permitiendo organizar y dar estructura y diseño. En ellos podemos insertar contenido HTML.

Etiquetas y atributos más usados

Incluyen los atributos estándares básicos: entre los que se encuentran accesskey, class, contenteditable, contextmenu, data, dir, hidden (escondido), id, lang, style (estilo), tabindex, títle

Inspección de una web

Podemos visualizar la estructura HTML de cualquier página que veamos en la we, solo pulsando en la pestaña "elements".

Textos

Para usar la función de subrayado usamos subrayado (u) y para usar caracteres, se usa la etiqueta code for (;;) { ... }. Finalmente, este se usa para inluir un enlace link.
