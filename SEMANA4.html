Semana 4

Puesta en operación del sitio web creado.

Procedimiento para creación de un sitio web

Lás páginas se componen por un sitio web, donde partiendo de una necesidad o un deseo, se realiza un documento para crear un mapa de sitio, mediante bocetos. Esto se lleva en lenguaje HTML a una página web, que funciona de manera circular, haciendo prueba y error para ver cómo va funcionando.

Nombre de dominio

La dirección donde se alojará la página creada dentro del servidor, tiene una estructura, comenzando por el protocolo, que incluye el https:// seguido del subdominio, el nombre de la página, la extensión, que puede ser .com o ,net, el código del país (que no siempre se incluye) y dividido por un slash se presentan los archivos que están dentro de la página.

Recursos escenciales

Lista de verificación

Lista de verificación. Compuesta por el contenido a publicar, donde se define la forma de la estructura del contenido a presentar en el sitio web. Luego el alojamiento web, donde se definen las características funcionales requeridas como el software, el soporte de sistemas complementarios, la base de datos, entre otros. Y por último el nombre del dominio, el cual será usado para el sitio web y sus características.

Alojamiento web

Conocido como web hosting. Existen diferentes tipos, como el hosting compartido, el hosting VPS, hosting en la nube (cloud hosting) y el hosting dedicado, siendo est último el más completo y costoso.

Alojamiento web

Dentro de sus principales criterios de selección existen: los costos o capacidad de almacenamiento, el panel de control, el tráfico mensual y la cantidad de conexiones, y además los servicios complementarios, como el soporte aplicaciones, CMS, certificaciones SSL.
